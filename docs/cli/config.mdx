# crewship config

Manage CLI configuration stored in `~/.crewship/cli-config.yaml`.

```bash
crewship config <subcommand>
```

## Subcommands

| Command | Description |
|---------|-------------|
| `show` | Display current configuration |
| `set` | Set a configuration value |

---

## `crewship config show`

Display current CLI configuration including config file path, server, workspace, format, and token status.

```bash
crewship config show
```

---

## `crewship config set`

Set a configuration value.

```bash
crewship config set <key> <value>
```

**Available keys:**

| Key | Description |
|-----|-------------|
| `server` | Server URL (e.g., `http://10.0.0.1:8080`). |
| `workspace` | Default workspace slug or ID. |
| `format` | Default output format: `table`, `json`, `yaml`, or `quiet`. |

```bash
crewship config set server http://localhost:8081
crewship config set workspace my-workspace
crewship config set format json
```

# crewship workspace

Manage workspaces -- isolated environments containing crews, agents, and configurations.

```bash
crewship workspace <subcommand> [flags]
```

**Alias:** `crewship ws`

## Subcommands

| Command | Description |
|---------|-------------|
| `list` | List all workspaces |
| `use` | Set the default workspace |
| `get` | Show workspace details |
| `create` | Create a new workspace |
| `update` | Update the current workspace |
| `member list` | List workspace members |
| `member add` | Add a member to the workspace |
| `member remove` | Remove a member from the workspace |
| `invite list` | List pending workspace invitations |
| `invite create` | Invite a user to the workspace by email |

---

## `crewship workspace list`

List all workspaces accessible to the current user. The active workspace is marked with `*`.

```bash
crewship workspace list
```

**Output columns:** SLUG, NAME, ID, ROLE

---

## `crewship workspace use`

Set the default workspace for all subsequent commands.

```bash
crewship workspace use <slug-or-id>
```

Validates that the workspace exists and is accessible before saving.

---

## `crewship workspace get`

Show workspace details.

```bash
crewship workspace get [slug-or-id]
```

If no argument is provided, shows the current default workspace.

**Output fields:** Name, Slug, Language, ID, Created.

---

## `crewship workspace create`

Create a new workspace.

```bash
crewship workspace create --name "My Workspace"
crewship workspace create --name "My Workspace" --slug my-ws --language cs
```

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--name` | `string` | _(required)_ | Workspace name. |
| `--slug` | `string` | _(auto from name)_ | Workspace slug. |
| `--language` | `string` | | Preferred language (e.g., `cs`, `en`). |

---

## `crewship workspace update`

Update the current workspace. Only changed flags are sent.

```bash
crewship workspace update --name "New Name"
crewship workspace update --language en
```

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--name` | `string` | | Workspace name. |
| `--slug` | `string` | | Workspace slug. |
| `--language` | `string` | | Preferred language (e.g., `cs`, `en`). |

---

## Workspace Members

### `crewship workspace member list`

List workspace members.

```bash
crewship workspace member list
```

**Output columns:** ID, EMAIL, NAME, ROLE, JOINED

### `crewship workspace member add`

Add a member to the workspace.

```bash
crewship workspace member add <user-id>
crewship workspace member add <user-id> --role ADMIN
```

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--role` | `string` | `MEMBER` | Role: `MEMBER` or `ADMIN`. |

### `crewship workspace member remove`

Remove a member from the workspace. Prompts for confirmation.

```bash
crewship workspace member remove <user-id>
crewship workspace member remove <user-id> --yes
```

| Flag | Short | Type | Default | Description |
|------|-------|------|---------|-------------|
| `--yes` | `-y` | `bool` | `false` | Skip confirmation prompt. |

---

## Workspace Invitations

### `crewship workspace invite list`

List pending workspace invitations.

```bash
crewship workspace invite list
```

**Output columns:** ID, EMAIL, ROLE, EXPIRES, CREATED

### `crewship workspace invite create`

Invite a user to the workspace by email.

```bash
crewship workspace invite create <email>
crewship workspace invite create user@example.com --role ADMIN
```

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--role` | `string` | `MEMBER` | Role: `MEMBER` or `ADMIN`. |
