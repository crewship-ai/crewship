Search

One box: matches prn_… id, routine slug, issue identifier, agent slug. Cmd-K opens the global picker.

Filter

Linear pattern: + Add filter → pick dimension → pick value → chip. Combine N filters; Saved views icon top-right persists combos per user.

Group

Group axis: Source / Routine / Crew / Issue / None. Default Source collapses cron noise. Persisted per user.

Scale

React-window virtualization on the flat row list. Group children paginate "Show N more" so a 1000-run cron stream stays scrollable.
