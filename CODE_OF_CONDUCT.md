# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its
code of conduct. By participating in this project — opening issues,
submitting pull requests, posting in discussions, or interacting with
maintainers and other contributors — you agree to abide by it.

The full text is available at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>

## Reporting

If you experience or witness behavior that violates the Code of Conduct,
please report it confidentially to **conduct@crewship.ai**. Reports are
read only by the project lead and are handled per the Enforcement
Guidelines in the Covenant.

Security vulnerabilities should be reported separately via the process
in [SECURITY.md](SECURITY.md).

## Scope

The Code of Conduct applies in all project spaces — the GitHub
repository, issue and pull-request discussions, the Discord/community
channels, and any other venue where someone represents the project
or interacts with its contributors.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
