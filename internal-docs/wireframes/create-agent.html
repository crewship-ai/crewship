My read

  • A (wizard) je nejlepší pro zachycení nováčků a stále rychlé pro power-userů (Skip + Enter). Nese nejvíc edukace ("SOUL" badge, persona = system_prompt).
  • B (tall form) je nejrychlejší pro někoho, kdo přesně ví, co dělá. Vizuálně náročnější.
  • C (template-first) je nejlepší jako vstupní bod, ale "Build custom" musí někam vést — typicky do B. Dohromady = "C → fallback to B" je pravděpodobně reálná final podoba.

Říkáš, kterou jdeme stavět — A, B, nebo kombo C+B?
