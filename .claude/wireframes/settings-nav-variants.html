Variant C — Recommended
Uplne cistý — zadne logo, max prostor
Zadne Crewship logo nikde v UI (jen favicon a login page). Workspace switcher v sidebar header, User v sidebar footer. Toolbar je ultra-cisty — jen page title + status + search + notifications. Maximalni vyuziti prostoru. Toto je pristup ktery pouziva Linear, Notion, Figma — produkt je znamy svym UI, ne logem v rohu.
